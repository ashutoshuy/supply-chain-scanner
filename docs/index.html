Quick Start

GitLab Scanning

supply-chain-scanner \
  --provider gitlab \
  --token glpat-xxx \
  --format json

GitHub Scanning

supply-chain-scanner \
  --provider github \
  --token ghp-xxx \
  --format csv

Integration Examples

GitHub Actions

- name: Supply Chain Scan
  run: |
    supply-chain-scanner \
      --provider github \
      --token ${{ secrets.GITHUB_TOKEN }}

GitLab CI

security_scan:
  script:
    - supply-chain-scanner 
        --provider gitlab 
        --token $GITLAB_TOKEN
